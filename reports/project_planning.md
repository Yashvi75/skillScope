
# Smart Resume Market Intelligence

## Project Scope

Target Domain:
AI and Machine Learning Job Market

Target Roles:
- AI Engineer
- Machine Learning Engineer
- GenAI Engineer
- Prompt Engineer
- LLM Engineer
- Computer Vision Engineer
- NLP Engineer
- AI Research Engineer
- MLOps Engineer
- Data Scientist
- Applied Scientist

Target Geography:
India (Primary Focus)

Target Dataset Size:
5000 Job Postings

Primary Objective:
Identify emerging AI roles, trending skills, and hidden skill combinations from real-world job postings.

## Raw Dataset Schema

Required Fields

- job_id
- job_title
- company
- location
- experience
- posted_date
- job_description

Optional Fields

- salary
- employment_type
- source
- scraped_at
